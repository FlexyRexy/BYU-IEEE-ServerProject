# BYU-IEEE-ServerProject

Scripts to run after installing Proxmox Virtual Environment

Copy and run this script in Node->Shell:
bash -c "$(curl -fsSL https://github.com/FlexyRexy/BYU-IEEE-ServerProject/blob/main/proxmox-post-install.sh)"

Copy and run this script in LXC->Console:
bash -c "$(curl -fsSL https://github.com/FlexyRexy/BYU-IEEE-ServerProject/blob/main/tailscale-subnet.sh)"
